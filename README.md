# langchain-goodmem

`langchain-goodmem` lets LangChain apps read from and write to GoodMem.
It covers the main workflows this package supports: connect to a GoodMem
space, manage the GoodMem resources needed for RAG/search, add content,
run semantic search, and optionally use one compatible GoodMem embedder as a
LangChain `Embeddings` object.
